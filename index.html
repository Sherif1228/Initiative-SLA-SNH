SNH Digital SLA Project

Project Establishment for the Future Supply Network Hub

Project Description & Shared Objective

To develop a Digital SLA APP that will empower users with a tool to evaluate the supply chain consequences of product initiatives. It will serve as a central hub for building scenarios, generating detailed reports, and consolidating findings for easier decision-making.

Project Overview: 3 Phases

Phase 1: Initial Scope of Deliverables

This phase focuses on creating the core tool with four main tabs: Scenario Creation, Line Up Details, Safety Stock Analysis, and an Initiative Summary.

Phase 1: Scenario Creation - Mockup Area

Phase 2: Enhanced Analytics

This phase enhances analytical capabilities by introducing weekly projection graphs for key performance indicators, including an Inventory Projection Tab and a TOF Projection Tab.

Phase 2: Analytics & Projections - Mockup Area

Phase 3: Scorecard

The final phase introduces a scorecard to show the status of different categories per market, providing a high-level overview for stakeholders.

Phase 3: Scorecard - Mockup Area

Team & Timeline

The project is scheduled to be completed by the end of September 2026, delivered in progressive waves by the Europe Touchless SLA team.
